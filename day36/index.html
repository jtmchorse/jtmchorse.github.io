Headless CMS

In this example we'll be using contentfulA as our Headless CMS. What is a headless cms? Lets assume we know what a CMS is. We can sum that up as a place to put content, and have that content rendered into a website. I'm sure we have all heard of Wordpress and maybe even Drupal. I hope you've never heard of Joomla.

A headless CMS it just a place to add contnet, nothing gets rendered. The CMS does expect to be sending that information around somewhere, likely through an API. So, we'll set that up to pipe the data into Gatsby.
